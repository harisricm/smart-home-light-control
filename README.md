## Project Overview

The Smart Home Light Control project is an IoT-based home automation system that allows users to control lights remotely. The system uses an ESP32 microcontroller and Wi-Fi connectivity to manage lighting devices efficiently.

## Project Scope

This project demonstrates how IoT technology can be used for home automation. Users can switch lights ON and OFF remotely, improving convenience and energy management.

## Objectives

* Control lights remotely
* Learn ESP32 programming
* Understand IoT communication concepts
* Develop a simple home automation solution
