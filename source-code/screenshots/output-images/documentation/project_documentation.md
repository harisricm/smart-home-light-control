# Smart Home Light Control

## Introduction

Smart Home Light Control is an IoT-based project that allows users to control lights remotely using an ESP32 microcontroller and Wi-Fi connectivity.

## Objective

The objective of this project is to automate light control and demonstrate the basic concepts of IoT and home automation.

## Components Used

* ESP32
* LED
* Wi-Fi Network
* Arduino IDE
* Wokwi Simulator

## Working Principle

1. ESP32 connects to a Wi-Fi network.
2. A web server is created on the ESP32.
3. Users access the web page through a browser.
4. ON/OFF buttons are used to control LEDs.
5. The ESP32 updates the LED status instantly.

## Advantages

* Easy remote control
* Energy saving
* Home automation
* Low-cost implementation

## Applications

* Smart homes
* Offices
* Smart buildings
* Energy management systems

## Conclusion

The Smart Home Light Control project demonstrates how IoT technology can be used to automate household appliances. The project provides a simple and effective way to control lighting remotely.
